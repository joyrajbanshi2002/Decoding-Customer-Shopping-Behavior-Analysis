# Decoding Customer-Shopping-Behavior-Analysis

This project analyzes customer shopping behavior using Python, PostgreSQL, and Power BI to uncover insights from 3,900+ customer transactions. The analysis focuses on customer demographics, purchase patterns, product preferences, discount usage, subscription behavior, and revenue trends.

Tools & Technologies
Python (Pandas, Data Cleaning, Feature Engineering)
PostgreSQL (Business Query Analysis)
Power BI (Interactive Dashboard & Visualization)
Key Features
Data cleaning and preprocessing
Missing value handling and feature engineering
SQL-based business analysis
Customer segmentation (New, Returning, Loyal)
Revenue and product performance analysis
Interactive Power BI dashboard
Business Insights
Revenue comparison by gender and age groups
High-spending discount users analysis
Subscriber vs non-subscriber spending behavior
Top-rated and most-purchased products
Customer loyalty and repeat purchase analysis
Goal

The project helps businesses understand customer behavior, improve marketing strategies, optimize discount policies, and enhance customer retention through data-driven insights.
